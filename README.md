GitHub como funciona
 O Git é uma ferramenta de controle de versao distribuida 
 essencial para gerenciar e registrar alterações em
 arquivos de projetos ele é um sistema de versão 
 Git hub pode armazenar arquivos que possuem comandos e códigos 
 Git é uma ferramenta poderosa e necessaria para qualquer desenvolvedor
 ele ajuda vocẽ criar alteraçoes em seu codigo,vocẽ pode salvar um snapshot de 
 seu projeto em um determinado ponto no tempo e fazer alteraçoes experimentais
 sem arriscar seu trabalho,ja que voce sempre pode voltar ao snapshot
